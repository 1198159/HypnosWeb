About Hypnos

*Please Note: This Website Is Still Under Development*

About Hypnos: Hypnos is a new 1.12.2 Carpet Mod Technical server that is focusing on finding new and innovative ways to accomplish tasks in minecraft and of course, have a good time. We are a more casual technical server that focuses on having fun while still expressing the technical aspect. Hypnos is predominantly a vanilla server running Carpet Mod 1.12.2 with some modifications to allow farmable resources instead of duping, modified carpet rules will be listed in the Hypnos Server Configuration below. The server relies on the community in order to make decisions and allow development of concepts and contraptions.
