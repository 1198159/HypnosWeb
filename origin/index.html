Hypnos Origin

*Please Note: This Website Is Still Under Development*

The Hypnos Server started November 2nd, 2019, the server was created with the idea of, trying new things in an old game. Hypnos is a relatively new technical server with the goal of finding new techniques to complete tasks in Minecraft. There are already well established farms and designs, we are looking for ways to improve lag and increase effciency. The name Hypnos originates from the Greek God of Sleep, as that is something no one gets enough of, in mythology Hypnos was able to overpower even Zeus and put him in a nap. If you would like to learn more about the actual minecraft server please visit the Discord server in the community tab.
